# Add-On Codes

> Illustrative reference text for demonstration. Not authoritative or current.

## What an add-on code is

An add-on code describes a service that is always performed in addition to a
primary procedure. Add-on codes are never reported alone. They are reported in
addition to the primary service code and describe additional work such as each
additional lesion, each additional hour, or each additional anatomic site.

## Reporting rules

An add-on code must be reported with an acceptable primary procedure code on the
same date of service by the same provider. If the primary procedure is not present
or not payable, the add-on code is denied. Add-on codes are generally exempt from
multiple-procedure payment reductions because the additional work is already
reflected in the code.

## Modifiers

Add-on codes are typically exempt from modifier 51, which identifies multiple
procedures, because they are not subject to the multiple-procedure reduction.
Appending modifier 51 to an add-on code is a common reporting error.

## Examples of the pattern

A base procedure reports the first unit of work, and a separate add-on code reports
each additional unit. For instance, a destruction code may report the first lesion
while an add-on code reports each additional lesion treated in the same session.
The add-on line cannot stand without its base line on the claim.
