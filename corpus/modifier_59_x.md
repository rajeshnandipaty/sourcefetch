# Modifier 59 and the X{EPSU} Modifiers — Distinct Procedural Service

> Illustrative reference text for demonstration. Not authoritative or current.

## Purpose

Modifier 59 identifies a procedure or service that was distinct or independent
from other non-E/M services performed on the same day. It is used to indicate that
two services which are normally bundled were, in this instance, separate. Because
modifier 59 has been overused, more specific X modifiers were introduced and are
preferred where they apply.

## The X modifiers

The X modifiers subdivide the situations that modifier 59 once covered on its own:

- XE, separate encounter: a service distinct because it occurred during a separate
  encounter on the same date.
- XS, separate structure: a service distinct because it was performed on a separate
  organ or anatomic structure.
- XP, separate practitioner: a service distinct because it was performed by a
  different practitioner.
- XU, unusual non-overlapping service: a service that is distinct because it does
  not overlap the usual components of the main service.

## When to use them

These modifiers should be applied to the Column Two code of an indicator-1 PTP edit
only when the medical record establishes that the services were genuinely separate.
Where a specific X modifier accurately describes the distinction, it is preferred
over the general modifier 59. The modifier does not by itself justify payment; the
documentation must support the distinct service.
